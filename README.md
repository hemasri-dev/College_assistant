# College Assistant Chatbot

A simple and attractive AI-inspired chatbot built with Python Flask, HTML, CSS, and Vanilla JavaScript. It helps students get instant information about college-related queries such as courses, fees, admissions, placements, hostel facilities, and more.

## Features

- Modern dark/light theme toggle.
- Real-time chat interface.
- Responsive design.
- Python Flask backend handling chatbot logic based on keywords.

## Setup Instructions

1. **Install Python**: Ensure you have Python installed on your system.
2. **Install Dependencies**: Open your terminal in this directory (`college_assistant`) and run:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the Application**: Start the Flask server by running:
   ```bash
   python app.py
   ```
4. **View in Browser**: Open your web browser and go to `http://localhost:5000`.

## Project Structure

- `app.py`: Main Flask server logic and dictionary data.
- `templates/index.html`: Main chatbot UI structure.
- `static/style.css`: Chatbot styling and themes.
- `static/script.js`: Client-side logic for message sending and API fetching.

## Usage

Interact with the bot by typing questions related to:
- Courses/Programs
- Admissions
- Fees
- Placements
- Hostels
- Departments
- Contact details
